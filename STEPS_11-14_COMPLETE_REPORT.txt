HOW TO USE THE TRAINED MODEL
================================================================================

Python Inference Code:

    import tensorflow as tf
    import numpy as np
    from tensorflow.keras.preprocessing.image import load_img, img_to_array
    import json
    
    # Load model
    model = tf.keras.models.load_model('best_model_transfer.h5')
    
    # Load class mapping
    with open('class_mapping.json', 'r') as f:
        class_mapping = json.load(f)
    
    # Load and preprocess image
    img = load_img('image.jpg', target_size=(224, 224))
    img_array = img_to_array(img) / 255.0
    img_batch = np.expand_dims(img_array, 0)
    
    # Make prediction
    probabilities = model.predict(img_batch)[0]
    predicted_class_idx = np.argmax(probabilities)
    predicted_class = class_mapping[str(predicted_class_idx)]
    confidence = probabilities[predicted_class_idx]
    
    print(f"Predicted: {predicted_class} (Confidence: {confidence:.2%})")
    
    # Get all class probabilities
    for idx, prob in enumerate(probabilities):
        print(f"  {class_mapping[str(idx)]}: {prob:.4f}")

Advanced Options:
  • Use tf.lite.TFLiteConverter to convert to TFLite for mobile
  • Use tf.keras.mixed_precision for faster inference on GPU
  • Use model.predict() with batch_size parameter for faster batch processing
